# Puissance 5

Application mobile de controle qualite terrain pour la gestion des sites, controles, incidents et rapports.

## Prerequis

- Node.js 20+
- `pnpm`

## Installation

```bash
pnpm install
```

## Lancement

```bash
pnpm start -- --clear
```

## Raccourcis utiles

```bash
pnpm ios
pnpm android
pnpm web
pnpm run typecheck
pnpm run lint
pnpm run test
```
