# Puissance 5

Application mobile de controle qualite terrain pour la gestion des sites, controles, incidents et rapports.

## Prerequis

- Node.js 20 LTS (>= 20.19.0 et < 21)
- `pnpm`
- Expo Go a jour (SDK 55)

## Installation

```bash
pnpm install
```

## Lancement

```bash
pnpm start -- --clear
```

Si tu vois une erreur "Project is incompatible with this version of Expo Go", mets Expo Go a jour depuis l'App Store / Play Store.

## Raccourcis utiles

```bash
pnpm ios
pnpm android
pnpm web
pnpm run typecheck
pnpm run lint
pnpm run test
```
